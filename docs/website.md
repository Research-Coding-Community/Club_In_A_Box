# Club Website

A club website gives people one stable place to find the schedule, join routes, resources, and contact information.

## Recommended Template

You can use the Research Coding Community website template:

- Template repository: [Research-Coding-Community/Coding_Club_Template](https://github.com/Research-Coding-Community/Coding_Club_Template)
- Example club: [Research-Coding-Community/biocode_club](https://github.com/Research-Coding-Community/biocode_club)

Use that template for the public website and this repository for the operating model, session templates, facilitation materials, and evaluation resources.

## Minimum Website Content

Your website should include:

- Club name and purpose.
- Who the club is for.
- Upcoming events.
- How to join or attend.
- Contact route.
- Code of conduct or participation expectations.
- Past materials.
- Links to partner clubs or the wider community.

## Good Defaults

- Keep the home page current.
- Put dates and times in a clear timezone.
- Mark beginner-friendly sessions.
- Avoid hiding contact details inside images.
- Link to materials after each session.
- Include accessibility information on event pages.

## Copy To Adapt

Use the website page copy template to draft your first homepage, about page, and event listing.
