# Connecting Communities

Club In A Box can be used by one club, but it is especially useful when several clubs want to stay connected while keeping their own identity.

## Hub-Of-Hubs Model

A hub-of-hubs is a light coordination layer across existing groups. It can provide:

- Shared calendar.
- Shared onboarding route.
- Common facilitation resources.
- Cross-promotion.
- Leadership development.
- Termly showcases.
- A place to identify gaps and duplication.

The hub should not take ownership away from local clubs. It should reduce repeated work and make connection easier.

## Coordination Meetings

Run a short coordination meeting once per term with club leads. Suggested agenda:

- What has each club run recently?
- What is coming next?
- Where are people asking for support?
- Are there clashes or opportunities to collaborate?
- What resources should be shared more widely?
- Who is at risk of burnout?

## Shared Language

Agree common definitions for:

- Beginner-friendly.
- Clinic.
- Co-working.
- Tutorial.
- Showcase.
- Helper.
- Convenor.
- Open materials.

This makes it easier for attendees to choose the right session.
