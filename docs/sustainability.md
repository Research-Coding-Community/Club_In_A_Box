# Sustainability

A club is sustainable when useful work continues without depending on one exhausted organiser.

## Reduce Single Points Of Failure

- Document recurring tasks.
- Rotate facilitation.
- Keep shared passwords and platforms owned by a team or institution, not one person.
- Maintain a public schedule.
- Keep notes in a shared place.
- Pair new helpers with experienced facilitators.

## Use Lightweight Roles

Common roles:

- Convenor.
- Session facilitator.
- Helper.
- Notes lead.
- Calendar maintainer.
- Website maintainer.
- Inclusion contact.
- Evaluation lead.

Use the roles and rotation template to plan rotation.

## Build Recognition

Community work should count. Help contributors describe:

- Facilitation.
- Training development.
- Mentoring.
- Open-source contribution.
- Event coordination.
- Cross-disciplinary leadership.
- Accessibility and inclusion work.

Use the leadership reflection template after events or at the end of term.

## Decide What To Stop

Sustainability requires stopping work as well as starting it. Review:

- Sessions with low value.
- Administration that nobody uses.
- Platforms that duplicate effort.
- Formats that exclude key audiences.
- Work that depends on one person every time.
