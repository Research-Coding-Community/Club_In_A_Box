# Feedback Form

Use this after a session. Keep it short enough that people answer before leaving.

## Questions

1. How useful was this session?
   - Very useful
   - Useful
   - Somewhat useful
   - Not useful

2. What was the most useful part?

3. What should we change next time?

4. Did the session feel welcoming?
   - Yes
   - Mostly
   - Not sure
   - No

5. What topics or formats would you like next?

6. Would you like to help with a future session?
   - Yes
   - Maybe
   - Not now
