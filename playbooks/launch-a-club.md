# Playbook: Launch A Club

## Goal

Launch a small, useful club with clear purpose, visible sessions, and a route for new people to join.

## Steps

1. Map demand.
   - Talk to potential attendees.
   - Identify related clubs or networks.
   - Run the community survey if needed.

2. Define the first version.
   - Fill in the club canvas.
   - Draft the club charter.
   - Pick a session format that is realistic to run.

3. Find a minimum organising team.
   - Name a convenor, facilitator, helper, and notes lead.
   - Agree what each role does.
   - Decide when roles will rotate.

4. Create a public home.
   - Use an existing intranet page, GitHub repository, mailing list, or website.
   - For a reusable club website, see the website guidance.

5. Schedule the first three sessions.
   - Publish dates early.
   - Avoid overloading the launch with too many formats.
   - Make the first session beginner-friendly.

6. Communicate clearly.
   - Use the event announcement template.
   - Ask partner groups to share it.
   - Say exactly who the session is for.

7. Review after three sessions.
   - What worked?
   - Who attended?
   - Who was missing?
   - Can someone new take a role?
   - Should the format continue, change, or stop?

## First Month Checklist

- Club canvas completed.
- Charter published.
- Code of conduct linked.
- First three dates held.
- Shared notes location created.
- Feedback route ready.
- At least one helper identified.
- Next review date agreed.
