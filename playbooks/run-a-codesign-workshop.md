# Playbook: Run A Co-Design Workshop

## Goal

Bring members, organisers, and potential contributors together to improve the club model and package reusable learning.

## When To Use This

Run a co-design workshop when:

- A new club has completed its first few sessions.
- Several clubs want to coordinate.
- You need to understand barriers to participation.
- You want to turn local learning into open resources.
- Leadership needs to become more distributed.

## Suggested Agenda

| Time | Activity | Output |
| --- | --- | --- |
| 0:00 | Welcome, purpose, and participation expectations | Shared understanding |
| 0:10 | Community map review | Corrections and gaps |
| 0:25 | What is working? | Strengths |
| 0:45 | What is hard to join or sustain? | Barriers |
| 1:05 | Break | |
| 1:15 | Resource prioritisation | Ranked template or playbook needs |
| 1:40 | Leadership mapping | Roles, gaps, and rotation options |
| 2:00 | Actions and owners | Next steps |
| 2:15 | Feedback and close | Evaluation |

Adjust the timing for your context. A shorter version can be run in 60 minutes.

## Materials

- Co-design workshop plan.
- Community map.
- Sticky notes or shared whiteboard.
- Dot voting or simple ranking method.
- Current templates or guides.
- Feedback form.

## Outputs To Publish

Publish only non-sensitive outputs:

- Updated priorities.
- New or revised templates.
- Session topics.
- Shared calendar actions.
- Leadership rotation plan.
- Summary of lessons others can reuse.

## After The Workshop

- Share a short summary.
- Open issues for agreed improvements.
- Invite pull requests for template changes.
- Add actions to the roadmap.
- Confirm the next review date.
