# Playbook: Connect Clubs

## Goal

Create a light coordination layer across clubs so people can find activity, share resources, and collaborate.

## Map The Landscape

List:

- Existing clubs.
- Convenors or contacts.
- Topics.
- Audiences.
- Session cadence.
- Platforms.
- Gaps.
- Duplication.

## Create Shared Infrastructure

- Shared calendar.
- Common event tags.
- Shared notes location.
- Cross-promotion route.
- Mailing list or chat channel.
- Termly coordination meeting.
- Repository of reusable materials.

## Keep Local Ownership

Each club should keep its own voice, audience, and programme. The shared layer should help with:

- Visibility.
- Reuse.
- Collaboration.
- Succession planning.
- Institution-wide learning.

## Coordination Agenda

Use this once per term:

- What has each club run?
- What is coming up?
- What has been reused?
- Where do people need help?
- Are there access or inclusion concerns?
- Are leaders overloaded?
- What should be shared nationally or publicly?
