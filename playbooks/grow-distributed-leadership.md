# Playbook: Grow Distributed Leadership

## Goal

Make leadership visible, shared, and sustainable.

## Start With Small Roles

People are more likely to contribute if the first step is concrete:

- Welcome people at the door.
- Monitor online chat.
- Take notes.
- Demo one tool for five minutes.
- Help test materials.
- Add an event to the calendar.
- Write a short session summary.

## Create A Role Ladder

| Step | Example activity |
| --- | --- |
| Attend | Join sessions and ask questions |
| Help | Support a practical exercise or answer chat questions |
| Contribute | Share a short demo, notebook, or resource |
| Facilitate | Lead part or all of a session |
| Convene | Coordinate programme, roles, and partnerships |
| Connect | Support other clubs or cross-community activity |

## Support New Facilitators

- Pair new facilitators with experienced helpers.
- Provide a run sheet.
- Offer a rehearsal.
- Keep the topic small.
- Make feedback constructive.
- Recognise facilitation as professional development.

## Review Leadership Health

Every term, ask:

- How many people held visible roles?
- Which roles are still held by one person?
- Who could shadow a role next?
- Which tasks are invisible and need naming?
- How are contributors recognised?

Use the roles and rotation template and leadership reflection template.
