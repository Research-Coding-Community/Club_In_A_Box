# Code Of Conduct

This project is for people building welcoming technical communities. We expect participation here to reflect that goal.

## Expected Behaviour

- Be respectful and constructive.
- Make space for people with different experience levels.
- Explain acronyms and assumptions when asked.
- Credit other people's work.
- Accept corrections with care.
- Focus criticism on ideas, materials, and processes rather than people.

## Unacceptable Behaviour

- Harassment, intimidation, or personal attacks.
- Dismissive comments about someone's skill level, role, background, accent, identity, or discipline.
- Gatekeeping or deliberately exclusionary language.
- Publishing private, sensitive, or identifiable information without consent.
- Repeated disruption of discussion or contribution processes.

## Reporting

If you experience or notice unacceptable behaviour, contact the maintainers using the route listed in the repository or organisation profile. Maintainers should respond promptly, take reports seriously, and avoid exposing the reporter unnecessarily.

## Scope

This code of conduct applies to repository activity, linked discussions, issues, pull requests, and community spaces explicitly connected to this project.
